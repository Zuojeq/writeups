__Javascrypt__
===========

_John Hammond_ | _Thursday, December 10th, 2015_ 

 
> Tyrin Robotics Lab uses a special web site to encode their secret messages. Can you determine the value of the secret key? 


----------

So when I first clicked on the link to get onto the web page, I did mess with the web page a little bit. To me it looked like it was [base64]-ing my input, but I didn't see anything to [_decrypt_][encryption], so I moved on.

I viewed the source of the web page, and you can pretty easily see some [client-side] [JavaScript]. My eyes quickly moved to the `key` variable which had the value of 'flag_' with some returned value of a function.

Honestly I felt a little too lazy to mentally reverse-engineer the function, so I just copied the page (`Ctrl+S` to save the web page) and modified the [JavaScript] on my own computer with a text editor. Then I could just view my own local copy of the code that displays the calculated flag for me.

All I ended up doing was adding an [`alert()`][javascript:alert] function call to see the key.

The embedded [JavaScript] ended up looking just like this... (no change, but added [`alert`][javascript:alert] call)

```
var key; // Global variable. 

// Since the key is generated when the page 
// is loaded, no one will be able to steal it
// by looking at the source! This must be secure!
function generateKey() {
    var i = 1;
    var x = 172;
    var n = 5493;
    while (i <= 25) {
        x = (x * i) % n;
        i++;
    }
    key = "flag_" + Math.abs(x);
    alert(key);
}

generateKey();

// Encode the message using the 'key'
function encode() {                                                        
    var input = $("#inputmessage").val();
    var output = CryptoJS.AES.encrypt(input, key);
    $("#outputmessage").val(output);
}        
```

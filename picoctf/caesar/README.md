The encrypted text is: "qebpbzobqmxppmeoxpbfpmnezwrsraakxlqbguvdcgvkavseawh".
If the title of this challenge is "Caesar," I am assuming this is a Caesar cipher. 

You can brute force this with an easy online tool like [http://www.dcode.fr/caesar-cipher](http://www.dcode.fr/caesar-cipher/).

If you throw it at the website, it should be able to find that +23 shift
for the [caesar cipher], giving you a result "THESECRETPASSPHRASEISPQHCZUVUDDNAOTEJXYGFJYNDYVHDZK".

__Vortex :: Level 0__
================


_John Hammond_ | _Wednesday, December 30th, 2015_ 


> Your goal is to connect to port 5842 on vortex.labs.overthewire.org and read in 4 unsigned integers in host byte order. Add these integers together and send back the results to get a username and password for vortex1. This information can be used to log in using SSH.

> Note: vortex is on an 32bit x86 machine (meaning, a little endian architecture)


----------

Here is the [solution](vortex0.py) I ended up putting together for the first [Vortex] challenge.

I have found this [cool resource](https://www.youtube.com/watch?v=rDU35RjLctc), and I helped use it to guide my thinking of this first challenge. I knew how to get the integers out and sum them together, but I did not realize I needed to send the answer in also packed.

I use [Python]'s [`socket`][socket] module to connect to the server, and use the [`struct`][struct] module to take the binary data received and put it into a number that we can actually work with. This is a common technique for quickly converting stuff into [little-endian][endianness] -- so you should read up on both of these modules and play with them a bit, if you haven't.

```
#!/usr/bin/env python

import socket
import struct

s = socket.socket( socket.AF_INET, socket.SOCK_STREAM )
s.connect( ( 'vortex.labs.overthewire.org', 5842 ) )

ints = []
for i in range( 4 ):

	ints.append( struct.unpack("I", s.recv(4) )[0] )
	# I read in 4 because that is the size of an unsigned integer

total = str( struct.pack( "I", sum(ints) ))
s.send(total)
print s.recv(4096)
```

I think these passwords change, so you may need to run this on your own time to get a new password.

__The password for vortex1 is `Gq#qu3bF3`.__

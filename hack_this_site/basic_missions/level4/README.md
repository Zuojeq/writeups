----------


So there is nothing useful for us in the source... but I feel like we must
have to hijack the e-mail that is being sent. If all we have to do is click
the button and it shoots out an e-mail to a specific address, we must be able
to mess with that in some way.

I recommend doing this with [TamperData], an add-on for [Firefox]. You can 
start the add-on by the 'Tools' menu within [Firefox], and hit the 
'Start Tamper' button before you hit the button to send the email. From there
you can change the email address it sends it to to your own!

Check your email, and you should have received the reminder email.
